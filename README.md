# FIFO-AND-LIFO
FIFO:(primeiro a entrar, Primeiro a sair) o primeiro cliente a entrar numa fila de supermercado vai ser o primeiro a sair, a saindo de um cliente depende só da primeira pessoa a esta na fila assim sucessivamente.




LIFO:(Ultimo que entrar, Primeiro a sair) o ultimo produto que chegará nos estoques de uma loja de construção são os primeiros a serem comprados, esse tipo não é tão usado em lojas pelo mundo.
